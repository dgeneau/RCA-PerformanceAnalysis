# RCA Performance Analysis Hub

Data loading and processing of performance measures inlcuding: 

 - Live GPS analysis 
 - Post Processing GPS reporting


METRICS: 
- Live GPS analysis: 
    - Prognostic time. Definied as the percentage of the gold medal target speed. Averaged over the stroke. Boat class can be selected using the dropdown menu on the card for each sensor. 
    - Boat Speed. m/s
    - Boat Split time (mm:ss.ms)
    - Stroke Rate. Definied as time between acceleration valleys, displayed as stroke per minute. 

- GPS Post Processing
